mobx-react-counter
=====================

一个小演示版本：用React + MobX 做的计数器小应用，主要是用来试验一下mobX的状态管理。

* Mobx 中文官网：http://cn.mobx.js.org/


### Run the example

```
npm install
npm start
```

Browser should open automatically. Otherwise, navigate to the URL reported in the terminal
